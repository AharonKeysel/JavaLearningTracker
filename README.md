# Java Learning Tracker
Desktop application for tracking study progress. Built with Java Swing, featuring CRUD operations and JSON data persistence.

## Features

- Create, edit and delete study topics
- Store topic title, category, status, study hours and creation date
- Notes for each topic
- Separate Topics and Notes tabs
- Filter topics by category
- Filter topics by status
- JSON data persistence
- Auto-save and auto-load
- Java Swing user interface

## Future Features

### Short-term

- Search topics by title
- Combined filtering
- Statistics dashboard
- Sorting by columns
- UI improvements
- Keyboard shortcuts

### Long-term

- Spring Boot backend
- Database support
- User authentication
- Export to PDF / Excel
- Charts and analytics

## Purpose

This project was created to practice Java development, object-oriented programming, desktop application design, and working with persistent data.
